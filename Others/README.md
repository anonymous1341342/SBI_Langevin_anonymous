This folder contains additional simulation examples.

## Subfolders

- `binomial`: This subfolder contains the code for the plot of score error v.s. prior density in the Beta-Binomial example (Figure 9).
- `multinomial`: This subfolder contains the code for the multinomial example that demonstrates the benefit of the curvature penalty. Run the .ipynb file and the two .R files to obtain Figure 3 and 12. 
- `epidemic_5floor`: This subfolder contains the code for the stochastic epidemic model in Appendix D.4, under the setting of 5 floors.
- `epidemic_10floor`: This subfolder contains the code for the stochastic epidemic model in Appendix D.4, under the setting of 10 floors.
