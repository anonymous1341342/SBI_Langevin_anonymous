This folder contains the code for the experiments in the paper.

The main examples in Section 5 are organized in the following subfolders:

- `Queuing`
- `MonoReg`
- `SDEMEM`
- `SDEMEM_realdata`

Additional experiments are collected in the `Others` folder.

Each experiment folder contains its own `README.md` file with instructions on how to run the corresponding code.

# Requirements

The main examples use Python and require the following packages:

- `numpy`
- `pandas`
- `matplotlib`
- `scipy`
- `torch`
- `sbi`
- `pot`
- `tqdm`

Some plot code is written in R. Those scripts use packages including:

- `ggplot2`
- `dplyr`
- `tidyr`
- `MASS`
- `RColorBrewer`
- `patchwork`
